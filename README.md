### В директории проекта доступны следующие скрипты:

#### `npm run build`
Запуск производственной сборки.

#### `npm run start`
Запуск проекта в режиме разработки.\
Откройте [http://localhost:3000](http://localhost:3000) для просмотра в браузере.
